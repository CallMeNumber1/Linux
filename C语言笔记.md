#### 位运算

- & 过滤
- | 并
- ^ 判断各个位上1出现的奇偶性

```c
#include <stdio.h>

int main() {
    int a, b;
    while (scanf("%d%d", &a, &b) != EOF) {
        int c, d;
        while(b != 0) {        //b=0时代表没有需要进位的了
            c = a ^ b;
            d = (a & b) << 1;  //a&b代表需要进位的位
            a = c;
            b = d;             
        }
    printf("%d\n", a);
    }
    return 0;
}
```
