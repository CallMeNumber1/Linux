#### 浮点数比较

一般有两种方式,bc和awk

- bc

```bash
if [ `echo "0.5 > 0.3" | bc` -eq 1 ]; then echo "ok"; else echo "not ok"; fi; 
```

- awk

```bash
x=3.1;
y=3.2;
echo "$x $y" | awk '{if ($1 > $2) print $1; else print $2}'
```

#### 比较

- 整数比较

```bash
-eq        等于,如:if [ "$a" -eq "$b" ] 
-ne        不等于,如:if [ "$a" -ne "$b" ] 
-gt        大于,如:if [ "$a" -gt "$b" ] 
-ge        大于等于,如:if [ "$a" -ge "$b" ] 
-lt        小于,如:if [ "$a" -lt "$b" ] 
-le        小于等于,如:if [ "$a" -le "$b" ] 
<        小于(需要双括号),如:(("$a" < "$b")) 
<=        小于等于(需要双括号),如:(("$a" <= "$b")) 
>        大于(需要双括号),如:(("$a" > "$b")) 
>=        大于等于(需要双括号),如:(("$a" >= "$b")) 
```

- 字符串比较

```bash
=        等于,如:if [ "$a" = "$b" ] 
==        等于,如:if [ "$a" == "$b" ],与=等价
!=        不等于,如:if [ "$a" != "$b" ] 
<        小于,在ASCII字母顺序下.如: 
>        大于,在ASCII字母顺序下.如: 
-z        字符串为"null".就是长度为0. 
-n        字符串不为"null" 
```
